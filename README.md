# Week 2 — Data Analysis Foundations with pandas

## Overview

This project introduces the use of pandas for data analysis. The objective is to transition from manual data processing using loops and dictionaries to professional data analysis workflows using DataFrames.

## Concepts Learned

* DataFrame and Series
* Loading structured data
* Data inspection
* Data filtering
* Statistical analysis
* Boolean indexing
* Summary statistics

## Analysis Performed

* Loaded student records into a pandas DataFrame
* Used `df.info()` to inspect the dataset
* Used `df.describe()` to generate summary statistics
* Calculated class average using `df["average"].mean()`
* Identified students above and below class average
* Generated a summary table of key metrics

## Technologies Used

* Python
* pandas
* JSON

## Learning Outcome

This project improved my understanding of how pandas simplifies data analysis and prepares datasets for future Machine Learning and Artificial Intelligence workflows.
